The Project

What is ToolHlp?

This project started as a simple git commit message helper — a frustration with having to remember the correct multi-line git commit syntax every time. It grew into a full toolkit for everyday developer needs.

Every tool is built with vanilla web technologies — no frameworks, no npm, no build step. Just open and use. The zero-dependency philosophy keeps it fast, auditable, and easy to self-host.

Getting Started

ToolHlp is a 100% browser-based developer toolkit. There's nothing to install, no account to create, and no data is ever sent to a server. Simply open the site and start using any tool instantly.

All tools run entirely in your browser using vanilla HTML, CSS, and JavaScript. The only exception is the URL Metadata tool which makes requests to external metadata APIs on your behalf.

Navigation

Use the navbar at the top to jump between tools. On desktop, hover over Tools to open the mega-menu with all tools organized by category. On mobile, tap the hamburger icon to open the sidebar.

Every page uses hash-based routing — you can bookmark or share any tool page directly (e.g. toolhlp.app/#commit-generator), and the browser back/forward buttons work as expected.

Theme

Click the icon in the top-right to toggle between dark and light mode. Your preference is saved to localStorage and remembered across visits.

Git Tools

Commit Generator

Generates correctly escaped, multi-line git commit commands. Supports CMD (uses ^ continuation), PowerShell (uses backtick), and Bash/Zsh (uses \).

Follows the Conventional Commits spec. Select a type, enter a title, optionally add a scope and body lines, then toggle Breaking Change if needed.

Example generated CMD command
git add . && ^ git commit -m "feat(auth): add JWT token refresh" ^ -m "- Implement sliding window token expiry" ^ -m "- Add refresh endpoint /api/auth/refresh" ^ -m "- Update middleware to validate refresh tokens"

Git Log Visualizer

Paste the output of either of these commands into the text area and click Visualize:

git log --pretty=format:"%h|%s|%an|%ar|%D" --all # or simply: git log --oneline --all

Each commit row shows the hash, message, author, relative time, and any branch/tag references. Click a row to expand the full detail panel.

Image Tools

EXIF Remover

Uses the browser's Canvas API to redraw each image, which inherently strips all metadata. Supports JPEG, PNG, and WebP. The "Strip ALL" mode redraws at ~95% quality for JPEG; PNG output is lossless.

The "Date fields only" removal mode is a best-effort metadata edit — it cannot strip GPS data at the byte level without a WASM library. For maximum privacy, always use "Strip ALL" mode.

Batch Renamer

Upload files, drag to set the desired order, then configure the naming pattern:

  • Prefix — text before the number (e.g. photo_)
  • Separator — character between prefix and number
  • Padding — zero-pad digits (e.g. 001 vs 1)
  • Suffix / Extension — appended after the number

Click Apply Pattern to preview new names, then Download ZIP to get all renamed files in a single archive.

Dev Tools

URL Metadata

Uses a 4-engine fallback chain to extract Open Graph metadata from any URL: Microlink → Dub.co → JSONLink → AllOrigins (raw HTML parse). History of the last 8 fetched URLs is saved in localStorage.

JSON Formatter

Paste raw JSON and choose Format (pretty print with 2-space indent) or Minify. Validation errors show the exact line and column. Syntax highlighting colors: strings, numbers, booleans/null.

Regex Tester

Type a pattern (without the leading/trailing /) and select flags. All matches are highlighted live in the test area. Capture groups are color-coded and listed in the results panel. Click any item in the cheat sheet to append it to your pattern.

Keyboard Tips

Universal shortcuts

All standard browser clipboard shortcuts work everywhere: Ctrl+A to select all in a textarea, Ctrl+C to copy, Ctrl+V to paste. Tool-specific copy buttons use the Clipboard API and show a toast notification on success.

File drop zones

The EXIF Remover and Batch Renamer both support drag-and-drop from your file explorer directly onto the drop zone — no need to click the browse button.

Privacy

ToolHlp is designed around a privacy-first philosophy:

  • No analytics, no tracking scripts, no cookies
  • No user accounts — your identity is entirely anonymous
  • Image processing happens entirely via the Canvas API — pixels never leave the browser tab
  • JSON, regex, and git log data stay in memory only; nothing is stored server-side
  • URL history is stored only in your own localStorage — it never leaves your device

The URL Metadata tool sends the URL you provide to third-party metadata APIs (Microlink, Dub.co, etc.) to extract Open Graph data. This is the only tool that makes external network requests.
