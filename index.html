What is DevToolHub?

DevToolHub is a free, browser-based collection of utilities built for developers. Everything runs client-side — no backend, no accounts, no tracking.

The project started as a Git commit helper and grew into a hub with tools for image processing, URL metadata extraction, JSON formatting and regex testing.

More tools will be added regularly. The stack is intentionally simple: vanilla HTML, CSS, and JavaScript — fast to load, easy to contribute to.

Git Commit Generator

Generate properly formatted, multi-line git commit commands for CMD, PowerShell, and Bash/Zsh with conventional commit support.

1

Select commit type

Click a conventional commit type (feat, fix, docs…) to prefix your title automatically.

2

Enter the title

Write a short, imperative subject line (max 72 chars recommended).

3

Add description (optional)

Add context, bullet points, or breaking change info. Each new line becomes a separate -m flag.

4

Choose your shell

Switch between CMD, PowerShell, and Bash/Zsh. The command escapes special characters correctly for each shell.

5

Copy & paste

Hit the copy button and paste the full command into your terminal.

Image EXIF / Metadata Remover

Strip EXIF metadata — including GPS location, camera info, and creation dates — from images entirely in your browser. Nothing uploaded to any server.

1
