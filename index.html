Getting Started

Tool Hlp is a single-page application — no installs, no server required. Just open index.html in any modern browser and all tools are ready to use.

Navigation

Use the navbar on desktop (hover over Tools for the mega menu) or the sidebar on mobile. Every tool opens instantly without a page reload.

Dark / Light Mode

Toggle the theme using the button in the top-right corner. Your preference is saved automatically.

Serving locally

python -m http.server 3000   # Python 3
npx serve .                  # Node.js

Most tools work on file:// directly. The URL Metadata tool needs an internet connection to call external APIs.

Git Tools

Commit Generator

Builds multi-line git commit commands for CMD, PowerShell, and Bash/Zsh. Supports conventional commit types, scope, breaking-change flag, and character counter.

Log Visualizer

Paste git log output and see a color-coded commit timeline with branch refs, author, and relative timestamps. Click any row for full detail.

Recommended log format

git log --pretty=format:"%h|%s|%an|%ar|%D" --all

The visualizer also accepts plain git log --oneline output.

Conventional Commit types

feat · fix · docs · style · refactor · test · chore · perf · ci · build · revert

Image Tools

EXIF Remover

Strips GPS location, creation dates, and camera metadata from JPEG/PNG/WebP images using the HTML5 Canvas API. Two modes: strip all metadata, or remove date fields only.

Batch Renamer

Upload multiple images, drag to reorder, configure a naming pattern (prefix, separator, padding, suffix, extension), preview all new filenames, then download as a ZIP.

Notes

  • JPEG files retain ~95% quality after canvas redraw
  • PNG files are losslessly re-encoded
  • No images are uploaded to any server

Dev Tools

URL Metadata

Extracts Open Graph data using a 4-engine fallback: Microlink → Dub.co → JSONLink → raw DOM parse. Shows title, description, image, video, language, keywords, and favicon.

JSON Formatter

Prettify with 2-space indentation, minify to single line, or validate with error position (line + column). Syntax highlighting for keys, strings, numbers, booleans, and nulls.

Regex Tester

Live match highlighting as you type, full flag support (g i m s u y), capture group display, and a 16-item click-to-insert cheat sheet. 1000-match safety cap prevents hangs.

Privacy

Tool Hlp is designed to be privacy-first:

  • No analytics, no cookies, no telemetry
  • All image processing happens in your browser via Canvas API — images never leave your device
  • JSON, regex, and git log data never leave your browser
  • The URL Metadata tool makes requests to third-party APIs (Microlink, Dub.co, JSONLink, AllOrigins) — only the URL you type is shared with those services
  • localStorage stores only theme preference and URL scan history

What is Tool Hlp?

Tool Hlp is a free, browser-based SPA that bundles everyday developer utilities into one place. Everything runs client-side — no backend, no accounts, no tracking.

It started as a git commit helper and grew into a multi-tool hub. The design philosophy is simplicity: open the HTML file, use the tools, done.
